# Asteroids Game

A simple video game, based on the classic Asteroids. If you've never played before, you can take a look at this version of the game:

![Asteroids Game Preview](https://github.com/user-attachments/assets/c91461a7-7e34-450a-aaae-5e8a701ed527)

## Prerequisites

* **Python 3.10+ installed** (see the Bookbot project for help if you don't already have this)
* The **`uv`** project/package manager
* Access to a **Unix-like shell** (e.g., zsh or bash)
